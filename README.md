# Engleesh

Личный словарь для изучения английских слов с автоматическим переводом, транскрипцией и озвучкой. Десктопное приложение на Python + PyQt6 и веб-версия для GitHub Pages.

## Возможности

### Основные

- **Умный ввод** — вводите слово на английском или русском, приложение автоматически определяет язык и переводит
- **Транскрипция** — IPA-транскрипция через `eng-to-ipa` + Free Dictionary API
- **Озвучка** — кнопка для прослушивания произношения (gTTS на десктопе, Web Speech API в браузере)
- **Редактирование слов** — нажмите на слово для выделения, затем кликните на английское или русское для inline-редактирования. При изменении английского слова транскрипция пересчитывается автоматически

### Self-check режим

- **Два тумблера** — «Скрыть перевод» и «Скрыть английский»
- Оба выключены → показывается всё
- Включён «Скрыть перевод» → русские слова скрыты
- Включён «Скрыть английский» → английское слово + транскрипция скрыты
- Скрытые ячейки открываются по клику

### Избранное

- Отмечайте слова звёздочкой ★ — они появляются во вкладке «Избранное»

### Папки

- **Создание папок** — выделите слова кнопкой «Выбрать», затем через меню ☰ создайте папку
- **Добавление в папку** — выделите слова и добавьте в существующую папку через меню ☰
- **Вкладка «Папки»** — список всех папок с количеством слов
- **Просмотр папки** — откройте папку, чтобы видеть только её слова
- **Удаление из папки** — свайп влево удаляет слово из папки (не из базы)

### Свайп удаления

- Свайп влево на слове → появляется красная кнопка удаления
- В папках свайп удаляет слово из папки, в основном списке — из базы

### Интерфейс

- **Анимации** — плавное появление строк, переключение тумблеров, кнопок
- **Иконки** — SVG иконки для озвучки и удаления
- **Интерфейс на русском языке**

## Запуск

### Десктоп — автоматический (двойной клик)

| ОС | Файл |
|----|------|
| Linux | `run.sh` |
| macOS | `run_macos.command` |
| Windows | `run.bat` |

Каждый лаунчер автоматически создаёт виртуальное окружение и устанавливает зависимости при первом запуске.

### Десктоп — ручной запуск

```bash
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
python app.py
```

### Веб-версия

Веб-версия расположена в папке `docs/` и может быть размещена на GitHub Pages. Откройте `docs/index.html` в браузере.

## Стек

### Десктоп

- **GUI:** PyQt6
- **Перевод:** deep-translator (Google Translate)
- **Транскрипция:** eng-to-ipa + Free Dictionary API
- **TTS:** gTTS (Google Text-to-Speech)
- **База данных:** SQLite3

### Веб

- **GUI:** HTML/CSS/JS (без фреймворков)
- **Перевод:** MyMemory API
- **Транскрипция:** Free Dictionary API
- **TTS:** Web Speech API
- **Хранение:** localStorage
