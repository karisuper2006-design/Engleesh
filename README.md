# Engleesh

Личный словарь для изучения английских слов с автоматическим переводом, транскрипцией и озвучкой. Кроссплатформенное десктопное приложение на Python + PyQt6.

## Возможности

- **Умный ввод** — вводите слово на английском или русском, приложение автоматически определяет язык и переводит
- **Транскрипция** — IPA-транскрипция через `eng-to-ipa` + Free Dictionary API в качестве запасного варианта
- **Озвучка** — кнопка 🔊 для прослушивания произношения (gTTS)
- **Режим самопроверки** — три варианта отображения: показать всё, скрыть перевод, скрыть оригинал. Скрытые ячейки открываются по клику
- **Система ошибок** — отмечайте сложные слова звёздочкой ★, они появляются во вкладке «Mistakes»
- **Persistent storage** — все слова сохраняются в SQLite (`dictionary.db`) и загружаются при перезапуске
- **Светлый дизайн** — чистый, минималистичный интерфейс

## Запуск

### Автоматический (двойной клик)

| ОС | Файл |
|----|------|
| Linux | `run.sh` |
| macOS | `run_macos.command` |
| Windows | `run.bat` |

Каждый лаунчер автоматически создаёт виртуальное окружение и устанавливает зависимости при первом запуске.

### Ручной запуск

```bash
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
python app.py
```

### Зависимости ОС для озвучки

| ОС | Команда воспроизведения |
|----|------------------------|
| macOS | `afplay` (встроенная) |
| Linux | `ffplay` (ffmpeg) или `mpv` |

Установка ffmpeg на Linux: `sudo apt install ffmpeg`

## Стек

- **GUI:** PyQt6
- **Перевод:** deep-translator (Google Translate)
- **Транскрипция:** eng-to-ipa + Free Dictionary API
- **TTS:** gTTS (Google Text-to-Speech)
- **База данных:** SQLite3
